# Support FAQ

## My integration stopped working suddenly. What do I check first?
1. Verify your API key hasn't expired (Settings -> API Keys).
2. Check the integration status page at status.clemta.com.
3. Re-authenticate the OAuth connection if the third party recently rotated credentials.
4. Review the activity log for the failing job's error code.

## How do I reset my password?
Click "Forgot password" on the login page. A reset link is sent within 60 seconds and is valid for 1 hour. If you don't receive it, check your spam folder.

## Is two-factor authentication available?
Yes, 2FA is available on all plans. Settings -> Security -> Enable 2FA. We support TOTP apps (Authy, Google Authenticator) and hardware keys (YubiKey).

## What's your data retention policy?
Active account data is retained indefinitely. After cancellation, data is held for 30 days for recovery, then permanently deleted. Audit logs are retained for 12 months.

## How do I export my data?
Settings -> Data -> Export. Choose CSV or JSON. Large exports run in the background and you receive an email with a download link when ready (typically within 1 hour).
